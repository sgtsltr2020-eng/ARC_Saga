# PHASE 5: DESKTOP UI (Planned)

> [!NOTE]
> Specs for the native Desktop Application.

## 1. Goal

Ship a native PyQt6 application to manage the ARC SAGA memory and orchestrator.

## 2. Tech Stack

- **GUI:** PyQt6 / PySide6
- **Packaging:** PyInstaller
- **Platform:** Windows 11 / macOS Big Sur+

## 3. Features

- **Token Dashboard:** Live usage vs Budget.
- **Orchestrator Panel:** Visual workflow builder.
- **Memory Explorer:** 3D graph view of the Knowledge Graph.
